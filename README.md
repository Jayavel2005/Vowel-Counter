# Vowel Counter Web App

This project is a simple web application that counts the number of vowels in a user-inputted text string.

## Features

- **Input Field**: Users can enter any text into the input field.
- **Count Button**: Clicking the "Count" button calculates and displays the number of vowels in the input text.
- **Reset Button**: Allows users to reset the vowel count to zero.
- **Styling**: Utilizes Bootstrap for a responsive and visually appealing layout.
- **Background**: Includes a custom background image to enhance the user interface.

## Technologies Used

- **Frontend**: HTML, CSS (Bootstrap)
- **Backend**: None (Pure client-side implementation)
- **JavaScript Libraries**: Bootstrap JavaScript, Popper.js (for Bootstrap tooltips)

## Usage

1. Enter any text into the "Enter the text" input field.
2. Click the "Count" button to see the number of vowels in the text.
3. Use the "Reset" button to clear the vowel count and start over.
